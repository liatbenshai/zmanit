# תיקון מלא - שישי/שבת כימי בית

## 📋 מה תוקן?

### 1. ✅ שישי ושבת = ימי בית מלאים
- שעות: 08:00 - 22:00 (14 שעות)
- כל המשימות נחשבות כמשימות בית
- ההתראות מחושבות מול זמן הבית

### 2. ✅ חישוב זמן נותר דינמי
- בסופ"ש: זמן נותר מחושב עד 22:00
- בימים רגילים: עד 16:15 (עבודה) / 21:00 (בית)

### 3. ✅ תצוגה דינמית
- כותרת: "שעות בית" במקום "שעות עבודה"
- סרגל שעות מתעדכן לפי היום
- התראות מותאמות

### 4. ✅ זיהוי משימות בית משופר
נוספו: shaked, dor, ava, yaron, cleaning, cooking, laundry
