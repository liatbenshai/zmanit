# תיקונים לזמנית - גרסה 9

## 🎯 מה חדש בגרסה זו:

### 1. ✅ פיצ'ר העברת משימות למחר - עובד!
מודל חדש שמאפשר:
- **לבחור** אילו משימות להעביר
- **לבחור תאריך**: מחר / מחרתיים / תאריך מותאם
- **סיכום** כמה משימות ודקות נבחרו
- **סימון בלוקים קבועים** (לא ניתנים להעברה)

### 2. ✅ תיקון תגיות עדיפות
עכשיו מוצג נכון:
- ⏰ **באיחור** - משימות שעבר הזמן שלהן (לא משנה מה העדיפות)
- 🔴 **דחוף** - משימות עם עדיפות דחופה (רק אם לא באיחור)
- 🟠 **גבוה** - משימות עם עדיפות גבוהה
- *(ללא תגית)* - משימות רגילות

### 3. ✅ עדיפות מועברת נכון
- ה-smartScheduler מעביר את העדיפות המקורית לבלוקים
- משימות שנוצרו כ"רגיל" יישארו רגילות
- משימות שנוצרו כ"דחוף" יישארו דחופות

### 4. ✅ טופס משופר
- תאריך התחלה + תאריך יעד (במקום שעה)
- ברירת מחדל: עדיפות רגילה

## 🚀 התקנה

1. **פתחי את ה-ZIP** והעתיקי את `src`, `public`
2. **ב-GitHub Desktop:** Commit + Push
3. **נקי cache:** F12 → Application → Clear site data
4. **רעננו:** Ctrl+Shift+R

## 📅 איך עובד מודל ההעברה

1. כשהזמן לא מספיק - לוחצים על **"ארגון מחדש - בחרי מה להעביר"**
2. מסמנים את המשימות שרוצים להעביר
3. בוחרים תאריך יעד (מחר / מחרתיים / תאריך אחר)
4. לוחצים **"העבר X משימות"**
5. המשימות מתעדכנות עם התאריך החדש!
