# זמנית - שדרוג מלא 🚀

## מה חדש בגרסה הזו

### 1. 🎨 דשבורד חדש ומרהיב!
- ציטוט מוטיבציוני יומי
- 🔥 סטריק - ספירת ימים רצופים
- טבעת התקדמות ויזואלית
- "משימה בוערת" - הכי דחוף עכשיו
- **📅 תכנון שבועי** - לחיצה על יום מוסיפה משימה לאותו יום

### 2. 🔄 גרירת משימות לשינוי סדר
- גררי משימה למעלה או למטה
- הסדר נשמר אוטומטית
- 3 פסים (☰) בצד = ידית גרירה

### 3. 📅 העברת משימות
בכל משימה יש כפתור 📅:
- מחר
- עוד יומיים
- עוד שבוע

### 4. 📊 מערכת למידה
- לומדת מכל משימה שמסיימים
- מציעה הערכות מדויקות יותר
- טאב "למידה" 🧠 בהגדרות

### 5. ✅ מודל משוב בסיום
- מציג הערכה מול ביצוע
- אפשרות לתקן את הזמן

### 6. 🏠 משימות בית (17:00-21:00)
- כביסה 🧺, קניות 🛒, ניקיון 🧹, בישולים 🍳

### 7. ⚙️ סוגי משימות מותאמים אישית
הגדרות > סוגי משימות
