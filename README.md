# תיקוני התראות - זמנית

## מה השתנה

✅ **התראות עובדות עכשיו!**
- 5 דקות לפני המשימה (ברירת מחדל)
- בדיוק בזמן המשימה
- צליל + פופאפ
- שמירה ב-localStorage (לא צריך Supabase)

## קבצים לעדכון

העתיקי את התיקיות `public` ו-`src` לתוך תיקיית הפרויקט והחליפי קבצים קיימים.

```
public/
  └── sw.js

src/
  ├── main.jsx
  ├── context/
  │   └── NotificationContext.jsx
  ├── pages/
  │   └── Settings.jsx
  ├── services/
  │   └── pushNotifications.js
  └── components/Notifications/
      └── SmartNotifications.jsx
```

## איך להפעיל

1. העתיקי את הקבצים
2. רעננו את הדף (Ctrl+Shift+R)
3. לכי ל**הגדרות** ⚙️
4. טאב **התראות** 🔔 (הראשון!)
5. לחצי **"אפשר התראות"**
6. לחצי **"בדיקה"** כדי לוודא שעובד

## הגדרות זמינות

|-------|------------|
| התראה לפני | 5 דקות |
| התראה בזמן | ✅ פעיל |
| צליל | ✅ פעיל |
