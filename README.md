# עדכון זמנית - 31/12/2024 (גרסה 6)

## 🐛 באגים שתוקנו

### 1. התראות על משימה פעילה ✅ תוקן עכשיו!
**הבעיה:** התראות "הגיע הזמן להתחיל" קופצות גם על המשימה שכבר עובדים עליה

**הפתרון:** 
- אם יש טיימר רץ על משימה - לא שולחים לה התראות "התחלה"
- בדיקה ישירה ב-localStorage לטיימר פעיל
- לוגים לדיבוג (פתחי Console לראות מה קורה)

### 2. התראות לפי לו"ז דינמי ✅
ההתראות מבוססות על הזמן המחושב, לא על הזמן המקורי

### 3. התראות על סיום משימה ✅
- התראה 5 דקות לפני סוף הזמן
- התראה כשהזמן נגמר
- התראה על חריגה

### 4. שאר התיקונים
- סדר בלוקים של פרויקטים (blockIndex)
- כפל זמן בעריכה
- התראות על משימות שהושלמו
- התראות על פרויקטים גדולים

## 💡 איך זה עובד

כשמשימה נדחית (כי עבדת יותר זמן על הקודמת), הלו"ז מחושב מחדש:

**מקור:** 
```
09:00 - משימה א (30 דק')
09:30 - משימה ב (30 דק')
10:00 - משימה ג (30 דק')
```

**אם עבדת על א' עד 09:45:**
```
09:45 - משימה ב (30 דק')  ← התראה תגיע ב-09:40!
10:20 - משימה ג (30 דק')  ← התראה תגיע ב-10:15!
```

ההתראות עוקבות אחרי הלו"ז הדינמי!
