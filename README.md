## 📤 איך להעלות ל-GitHub

### אפשרות 1: דרך GitHub (מומלץ)

1. לכי לריפו שלך ב-GitHub
2. לחצי על "Add file" → "Upload files"
3. גררי את **תיקיית src** לחלון
4. למטה כתבי: "עדכון: תיקון יומן גוגל + ניקוי קוד"
5. לחצי "Commit changes"

### אפשרות 2: החלפה ידנית

אם זה לא עובד, תעלי קובץ-קובץ:
1. לכי לכל קובץ ב-GitHub
2. לחצי על העיפרון ✏️
3. מחקי הכל, הדביקי את התוכן החדש
4. לחצי "Commit changes"
