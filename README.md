# זמנית - שדרוג מלא 🚀

## מה חדש בגרסה הזו

### 1. 📊 מערכת למידה אוטומטית
המערכת לומדת מכל משימה שמסיימים:
- כמה זמן הערכת
- כמה זמן לקח בפועל
- מציעה הערכות מדויקות יותר בפעם הבאה

### 2. ✅ מודל משוב בסיום משימה
כשמסיימים משימה קופץ מודל קטן:
- מציג הערכה מול ביצוע בפועל
- מאפשר לתקן את הזמן אם לא מדויק

### 3. 💡 הצעת הערכה חכמה
בטופס משימה חדשה:
- אם יש מספיק נתונים מהעבר
- מוצגת המלצה להתאמת הזמן

### 4. 📅 העברת משימות
בכל משימה יש כפתור 📅 שמאפשר:
- העברה למחר
- העברה לעוד יומיים
- העברה לעוד שבוע

### 5. 🏠 משימות בית (17:00-21:00)
סוגי משימות חדשים מובנים:
- כביסה 🧺
- קניות 🛒
- ניקיון 🧹
- בישולים 🍳

### 6. ⚙️ סוגי משימות מותאמים אישית
אפשר להוסיף סוגי משימות חדשים!
- בחירת אייקון
- בחירת קטגוריה (עבודה/משפחה/בית)
- הגדרת זמן ברירת מחדל

### 7. 🔔 תיקון התראות
לא מפריע כשעובדים על משימה אחרת

### 8. 📌 תיקון סדר משימות
משימה עם טיימר רץ נשארת ראשונה

## 🔧 איך להתקין

### שלב 1: החלפת קבצים
העתיקי את הקבצים הבאים ל-GitHub (החלפה):
- `src/config/taskTypes.js`
- `src/components/DailyView/DailyTaskCard.jsx`
- `src/components/DailyView/SimpleTaskForm.jsx`
- `src/components/Notifications/NotificationChecker.jsx`

### שלב 2: הוספת קבצים חדשים
צרי את הקבצים הבאים:
- `src/utils/taskLearning.js`
- `src/components/Settings/TaskTypeSettings.jsx`

### שלב 3: הוספת גישה לדף ההגדרות
ב-`App.jsx` או בתפריט הראשי, הוסיפי כפתור לפתיחת דף ההגדרות:

```jsx
import TaskTypeSettings from './components/Settings/TaskTypeSettings';

// בתוך הקומפוננטה:
const [showSettings, setShowSettings] = useState(false);

// בתפריט:
<button onClick={() => setShowSettings(true)}>
  ⚙️ הגדרות סוגי משימות
</button>

// בסוף:
{showSettings && (
  <TaskTypeSettings onClose={() => setShowSettings(false)} />
)}
```

## 💡 שימוש

### הוספת סוג משימה חדש
1. לחצי על ⚙️ בתפריט
2. בחרי קטגוריה (עבודה/משפחה/בית)
3. לחצי "+ הוסף סוג משימה חדש"
4. בחרי אייקון, שם, וזמן ברירת מחדל
5. לחצי "הוסף"

### העברת משימה למחר
1. בכרטיס המשימה, לחצי על 📅
2. בחרי "מחר" / "עוד יומיים" / "עוד שבוע"

### צפייה בסטטיסטיקות למידה
המערכת לומדת אוטומטית. ההצעות יופיעו בטופס משימה חדשה אחרי שתסיימי 3+ משימות מאותו סוג.
