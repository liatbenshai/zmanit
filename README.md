# זמנית - שדרוג מלא 🚀

## מה חדש בגרסה הזו

### 1. 📊 מערכת למידה אוטומטית
המערכת לומדת מכל משימה שמסיימים:
- כמה זמן הערכת
- כמה זמן לקח בפועל
- מציעה הערכות מדויקות יותר בפעם הבאה

### 2. ✅ מודל משוב בסיום משימה
כשמסיימים משימה קופץ מודל קטן:
- מציג הערכה מול ביצוע בפועל
- מאפשר לתקן את הזמן אם לא מדויק

### 3. 💡 הצעת הערכה חכמה
בטופס משימה חדשה:
- אם יש מספיק נתונים מהעבר
- מוצגת המלצה להתאמת הזמן

### 4. 📅 העברת משימות
בכל משימה יש כפתור 📅 שמאפשר:
- העברה למחר
- העברה לעוד יומיים
- העברה לעוד שבוע

### 5. 🏠 משימות בית (17:00-21:00)
סוגי משימות חדשים מובנים:
- כביסה 🧺
- קניות 🛒
- ניקיון 🧹
- בישולים 🍳

### 6. ⚙️ סוגי משימות מותאמים אישית
בדף הגדרות > סוגי משימות:
- בחירת אייקון
- בחירת קטגוריה (עבודה/משפחה/בית)
- הגדרת זמן ברירת מחדל

### 7. 🧠 טאב למידה חדש בהגדרות
צפייה בסטטיסטיקות:
- יחס הערכה מול ביצוע לכל סוג משימה
- מגמה (משתפרת/יורדת/יציבה)
- אפשרות למחוק נתונים

### 8. 🔔 תיקון התראות
לא מפריע כשעובדים על משימה אחרת

## 💡 שימוש

### הוספת סוג משימה חדש (כמו "חוג מקרמה")
1. לכי להגדרות (⚙️)
2. לחצי על טאב "סוגי משימות"
3. בחרי קטגוריה "בית"
4. לחצי "+ הוסף סוג משימה חדש"
5. בחרי אימוג'י (למשל 🎨)
6. כתבי "חוג מקרמה"
7. הגדירי זמן (למשל 120 דקות)
8. לחצי "הוסף"

### העברת משימה למחר
בכל משימה יש כפתור 📅 - לוחצים ובוחרים לאן להעביר.

### צפייה בסטטיסטיקות למידה
הגדרות > טאב "למידה" 🧠
