# עדכון זמנית - גרסה 4.2

## מה חדש בגרסה זו:
- ✅ **קטגוריות מותאמות אישית** - תמלול ותרגום קבועות, שאר לבחירה
- ✅ **אינטגרציה מלאה בהגדרות** - ניהול קטגוריות מופיע ישירות בדף ההגדרות
- ✅ תמיכה מלאה בנייד (החלקה בין ימים)
- ✅ שעות עבודה מותאמות לכל יום
- ✅ הצעות נדחות נזכרות לשבוע

## הוראות התקנה:

### שלב 1: העתקת קבצים
פתחי את ה-ZIP והעתיקי את תיקיית `src` לתוך הפרויקט שלך.
הקבצים יתווספו/יחליפו אוטומטית במיקומים הנכונים.

### שלב 2: בדיקה
רענני את האפליקציה (Ctrl+Shift+R)

## מבנה הקבצים:

```
src/
├── config/
│   └── taskCategories.js         ← קונפיגורציית קטגוריות
├── components/
│   ├── Planning/
│   │   └── WeeklyPlannerPro.jsx  ← תצוגה שבועית + מובייל
│   └── Settings/
│       ├── SettingsPage.jsx      ← מעודכן! + טאב קטגוריות
│       └── CategoryManager.jsx   ← ניהול קטגוריות מותאמות
└── utils/
    └── smartSchedulerV4.js       ← שעות עבודה מותאמות
```

## איך זה עובד:

### קטגוריות מותאמות:
1. לחצי על ⚙️ (הגדרות)
2. בחרי "🎨 קטגוריות מותאמות" מהתפריט
3. הוסיפי קטגוריות חדשות או בחרי מהרשימה המוצעת

### קטגוריות קבועות (לא ניתן למחיקה):
- 🎙️ תמלול
- 🌍 תרגום

### קטגוריות מוצעות להוספה:
- 📝 הגהה, 📧 מיילים, 📅 פגישה
- 💬 לקוחות, 👔 ניהול, 📚 קורס
- ועוד...

## בדיקה לאחר התקנה:
1. רענני את האפליקציה (Ctrl+Shift+R)
2. פתחי הגדרות ⚙️
3. ודאי שמופיע טאב "🎨 קטגוריות מותאמות"
4. נסי להוסיף קטגוריה חדשה
