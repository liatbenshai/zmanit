# 📅 הוספת יומן גוגל לזמנית

## 📦 מה בתיקייה:

```
index.html                          ← החלפה בתיקייה הראשית
src/hooks/useGoogleCalendar.js      ← קובץ חדש
src/components/DailyView/DailyView.jsx  ← החלפה
```

## 🚀 הוראות:

### שלב 1: גיבוי
לפני שמחליפים קבצים, תשמרי גיבוי של הקבצים הקיימים!

### שלב 2: החלפת קבצים
1. החליפי את `index.html` בתיקייה הראשית של הפרויקט
2. העתיקי את `useGoogleCalendar.js` לתיקייה `src/hooks/`
3. החליפי את `DailyView.jsx` בתיקייה `src/components/DailyView/`

### שלב 3: הפעלה
רענני את האתר (Ctrl + Shift + R)

## ✨ מה תקבלי:

כפתור "יומן גוגל" בפינה השמאלית למעלה עם:
- 📤 ייצא משימות ליומן - שולח את המשימות ליומן גוגל
- 📥 ייבא מיומן גוגל - מביא אירועים מגוגל לזמנית
- 🔌 ניתוק

## 🔧 פתרון בעיות:

אם הכפתור לא עובד:
1. בדקי שה-index.html עודכן (צריך להיות שם שורות עם google)
2. רענני את הדף
3. נסי לנקות cache (Ctrl + Shift + Delete)
